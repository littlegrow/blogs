### 服务器

<br>

#### 免输密码登录服务器

```
#!/usr/bin/expect
spawn ssh 用户@远程主机IP 
expect "password:"  
send "密码\r"
interact 
```

<br>

#### 办公网络传输文件至本地服务器

```
#!/usr/bin/expect
set file [lindex $argv 0]
spawn scp -r $file 用户@本地服务器IP:/tmp
expect "password: "
send "用户密码\r"
expect eof
```
